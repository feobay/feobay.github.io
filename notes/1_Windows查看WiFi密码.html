# Windows查看已经连接过的WiFi密码

转载自微信公众号 【青衣十三楼飞花堂】，作者：沈沉舟

// 确保wlansvc服务启动中，没有无线网络的台式机，该服务未启动
$ sc query wlansvc | findstr STATE

// 查看曾经连接过的无线网络
$ netsh wlan show profiles

// 若wlansvc服务未启动，上述命令报错
$ netsh wlan show profiles
The Wireless AutoConfig Service (wlansvc) is not running.

// 假设一切顺利，如下命令可查看指定无线网络的明文密码
// 如果是Windows 中文版，需要将 "Key Content" 替换为 "关键内容"
$ netsh wlan show profiles name="xxx" key=clear | findstr /C:"Key Content"
    Key Content            : XXXXXXXXXXXXXXXX

PS: 记得使用cmd，使用PowerShell部分命令无输出
